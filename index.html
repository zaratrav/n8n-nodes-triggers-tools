About n8n Nodes & Triggers

n8n nodes and triggers are the core building blocks of workflow automation, allowing you to define logic, handle data, and connect systems.

Triggers such as webhooks, cron jobs, and schedules start workflows based on events or time conditions.

Logic nodes like IF, Switch, and Merge control the flow of data and enable conditional execution.

Processing nodes such as Set, Loop, and Split In Batches help manipulate and scale data handling.

Code nodes (JavaScript and Python) allow custom logic, making n8n extremely flexible for advanced automation.

Expressions and data transformation enable dynamic workflows and powerful integrations across APIs and services.
